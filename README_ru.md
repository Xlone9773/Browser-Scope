Высококачественная и проработанная до мелочей утилита для глубокого анализа браузерного окружения и надежная диагностическая панель. Платформа предоставляет комплексные тесты производительности оборудования, сетевые утилиты и аудит безопасности.

## ✨ Ключевые преимущества

- 🔍 **Глубокий анализ возможностей** — Охватывает 13+ ключевых направлений, включая операционную систему, аппаратное обеспечение, сети, хранилища, политики безопасности, медиаустройства и веб-API.
- 🧰 **Универсальные сетевые утилиты** — Включает в себя тест скорости (Speed Test), точное определение IP-адресов и проверку связности по UDP Ping.
- 🎨 **Безупречный визуальный стиль** — Поддержка нескольких динамических тем, кастомная палитра цветов, управление анимациями и модульный дизайн на основе карт с акцентом на пустое пространство.
- 🌍 **Полноценная локализация** — Встроенная поддержка 6 языков (русский, английский, упрощенный/традиционный китайский, японский) с возможностью динамического переключения.
- 🗄️ **Полнофункциональный прокси-движок** — Работает на базе Express (Node.js), обеспечивая обход CORS, безопасное проксирование API-запросов и глубокое зондирование UDP-портов.
- 📜 **Лицензионная прозрачность (Attributions)** — Специальный информационный раздел с подробным перечнем используемых библиотек с открытым исходным кодом, распределением ролей и используемых шрифтов.

## 🚀 Развертывание и системные требования

> [!WARNING]
> **Full-Stack Архитектура:** Этот проект спроектирован как полноценное full-stack приложение, объединяющее клиентскую часть на React и серверную часть на Express (Node.js). Сервер необходим для обхода ограничений CORS, проксирования запросов и сетевых проверок. Вы **должны** развертывать его в среде Node.js (например, Cloud Run, Docker, VPS), а не как простой статический сайт.

> [!CAUTION]
> **Ограничения в Serverless-среде:** На многих облачных платформах (таких как Vercel, AWS Lambda или Edge-функции) исходящие UDP-сокеты и долгоживущие соединения могут быть полностью заблокированы.
> Приложение оснащено функцией **автоматического определения ограничений среды**: если UDP-проксирование не поддерживается платформой, соответствующие переключатели в настройках станут неактивными (серыми) с надписью «Не поддерживается». Для разблокировки всех сетевых возможностей рекомендуется использовать VPS или полноценные контейнеры (Docker/Cloud Run).

### Практические утилиты
- **Расширенная диагностика сети** — Высокоскоростное тестирование в обход CORS и UDP DNS-запросы для проверки живучести соединения.
- **Локальный ИИ (Transformers.js)** — Выполнение нейросетевых моделей непосредственно в браузере с помощью WebAssembly для бенчмаркинга CPU/GPU.
- **Экспорт PDF-отчетов высокого качества** — Послойный рендеринг структуры DOM с помощью `html2canvas` и `jsPDF` для генерации подробных диагностических карт.
- **Информационная панель (Attributions)** — Справочник по используемым зависимостям (React, FingerprintJS, Motion, Workbox) и гарнитурам шрифтов с указанием их авторов и лицензий.
- **Экспорт логов** — Сохранение сотен параметров диагностики в стандартизированном JSON-файле.
- **Плавающий монитор** — Компактный, перетаскиваемый виджет со статистикой производительности.
- **Поддержка PWA** — Возможность установки в качестве полноценного приложения для рабочего стола или мобильного экрана с поддержкой offline-режима.

## 📝 Руководство по запуску

### Локальная разработка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Xlone9773/Browser-Scope
cd Browser-Scope

# 2. Установите все зависимости (для фронтенда и сервера Express)
npm install

# 3. Запустите сервер разработки
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000) в браузере для просмотра и отладки приложения.

### Сборка и релиз в продакшн

```bash
# Сборка клиентского бандла и оптимизация сервера Express
npm run build

# Запуск скомпилированного full-stack приложения
npm run start
```

```text
/
├── appearance/          # Настройки тем и визуального отображения

│   ├── cards/           # Карты диагностики (система, процессор, сеть и т.д.)

│   ├── hardware/        # WebGL-профилирование и замеры GPU
│   ├── heatmap/         # Визуализации тепловых карт и датчиков движения
│   ├── layout/          # Базовые блоки интерфейса (шапка, подвал)
│   ├── sections/        # Разделительные группы и секции
│   ├── settings/        # Настройки и отладочная панель разработчика
│   ├── speedtest/       # Комплексный тест пропускной способности сети
│   └── ui/              # Библиотека атомарных UI-компонентов без состояния
├── hooks/               # Пользовательские хуки React
├── services/            # Фоновые аналитические модули
│   ├── detectors/       # Скрипты и тесты обнаружения параметров
│   ├── app.worker.ts    # Фоновый Web Worker для вычислительных нагрузок
│   ├── detectionService.ts # Единый диспетчер запуска диагностических тестов
│   └── score.ts         # Алгоритм оценки производительности устройства
├── utils/               # Общие вспомогательные утилиты
```
