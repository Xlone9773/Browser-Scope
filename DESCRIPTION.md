# BrowserScope - Project Overview

BrowserScope is a comprehensive browser fingerprinting and system intelligence platform built with React 19 and TypeScript. The application provides deep insights into browser capabilities, hardware specifications, security features, and AI readiness.

## Key Features

- **Advanced Fingerprinting**: Canvas, WebGL, and audio context analysis
- **Hardware Detection**: CPU, GPU, memory, battery, and peripheral information
- **Security Analysis**: Bot detection, privacy controls, and ad blocker identification
- **AI Compute Readiness**: JavaScript performance benchmarking for AI workloads
- **Modern Web API Detection**: Over 100+ browser features and capabilities
- **PWA Assessment**: Progressive Web App installation and capability evaluation
- **Media Codec Support**: Video, audio, and image format compatibility
- **Real-time Scoring**: Uniqueness assessment with detailed breakdowns

## Technology Stack

- React 19
- TypeScript 5.8
- Vite 6.2
- Tailwind CSS
- FingerprintJS
- @xenova/transformers
